# アーリーぷっしゃー！

様々な形式のクイズをサポートするためのソフトウェアです。

## 説明

`アーリーぷっしゃー！`はゲームパッドの各ボタンを早押しボタンと見立てて押されたときのプレイヤー表示
や排他制御、得点計算などをサポートします。  
複数のクイズ形式に対応しており、ルール毎に処理が異なります。

操作やハードについて詳しくはWikiをご覧ください。

## 動作環境

* Windows 7 以上
* .Net Framework 4.6.1 以上
* プレイヤー/ボタン数のゲームパッド
  * アナログスティックや方向キーはボタンに含まれません
